Instructions for using OMRAT application

Internet browser

Recommended browser: Google Chrome

Sheet music pictures and annotation files

To load sheet music pictures, use Open collection button from File dropdown menu. This lets to specify multiple files to allow switching them with ease in the application.

To load annotation files, use Open annotations button from File dropdown menu. This lets to specify multiple files in order to load them according to loaded sheet music pictures. The files should be in YOLO format or be files downloaded from the application. The files' names should match picture files' names, which they describe.

To download annotation files, use Download annotations button from File dropdown menu. This downloads file with annotations of currently displayed image.

To remove all annotations from currently displayed image, use Clear annotations from Edit dropdown menu.

To switch pictures from loaded collection, use Next or Previous button at the right panel at Images section.

To see any picture from loaded collection, click its title at the right panel at Images section.

Navigation and marking modes

Navigation mode switching buttons and current mode properties are located at the left panel.

Available navigation modes are:

Cursor
- allows to scroll over displayed picture and edit objects,
Mark
- allows to choose label, that will be bound to new objects and drawing those objects,
Group
- allows to choose label, that will be bound to new relations and drawing or removing those relations.

Selected object properties

Selected object properties are located at the left panel under current mode properties.

When an object is selected, it is possible for it to:

Label
- allows to choose the object's label,
Duplicate
- create a copy of the object, the object is pasted next to copied object,
Remove
- remove the object from the picture. If removed object has any relations, they are gone.
Next
- select next object of the same label. If does not exist, selects first such object.
Previous
- select previous object of the same label. If does not exist, selects last such object.

Navigation and annotating

Mouse

By rotating mouse wheel, it is possible to zoom the picture in and out.

In Cursor mode, by holding mouse key down on an object and dragging the pointer to a chosen place, the object is moved.

In Cursor or Group mode, by holding mouse key down and dragging the pointer to a chosen place on the image, the picture is scrolled.

In Cursor or Group mode, by clicking an object, it becomes selected.

In Cursor mode, by dragging a pulley around selected object, the object is resized.

In Mark mode, by holding mouse key down and dragging the pointer to a chosen place on the image, a rectangular object is drawn. By holding Ctrl, it is possible to draw square object.

In Group mode, by selecting another object, a relation between previously selected object and recently selected object is created. If the relation already exists it will be removed.

Keyboard

Selected objects can be moved by pressing arrow keys. Each key press moves the object by one pixel of currently displayed image.

Key shortcuts:

Duplicate
: Ctrl + D
Remove
: DELETE
Changing navigation mode to Cursor
: C
Changing navigation mode to Mark
: W
Changing navigation mode to Group
: G
Next picture
: D
Previous picture
: A
Download annotations
: Ctrl + S

User interface language change

To change user language, click flag located at top right corner of the site. Currently available languages are English and Polish.
